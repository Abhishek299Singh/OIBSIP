# FlowTask — Minimalist To-Do List Web Application

FlowTask is a clean, beautiful, and fully responsive task management web application built entirely using modern Vanilla HTML, CSS, and JavaScript. It helps users manage their daily intentions and stay organized with separate states for pending and completed tasks.

---

## 🌟 Features

- **Add Tasks**: Quickly jot down new intentions.
- **Toggle Completion**: Complete tasks to move them into the "Completed Tasks" list with a clean strikethrough styling, or toggle them back to "Pending".
- **Delete Tasks**: Remove items instantly from either list with a single click.
- **Persistent State**: Tasks are saved automatically using browser `localStorage`, ensuring data persists even after refreshing or closing the browser.
- **Premium Interface Design**:
  - Outfitted with modern typography (Outfit Font).
  - Responsive layouts fitting desktop, tablet, and mobile screens.
  - Interactive custom checkboxes with smooth SVG checkmark transitions.
  - Delightful micro-animations for adding, completing, and deleting tasks.
  - Built-in Dark Mode support matching your system theme configuration.

---

## 🛠️ Technologies Used

- **HTML5**: Semantic structural elements for SEO and accessibility.
- **CSS3**: Custom variables, flexbox, layout grids, keyframe animations, and automatic media-query based dark mode adaptation.
- **JavaScript (ES6+)**: Custom DOM manipulation, events, state tracking, and local storage integrations.
- **FontAwesome**: Scalable vector icons for high-definition rendering.

---

## 🚀 How to Run the Project

Since FlowTask is written using raw client-side technologies, running it is extremely simple:

1. **Locate the project files** on your computer.
2. **Double-click** the `index.html` file to open it in your default web browser (Chrome, Safari, Firefox, Edge, etc.).
3. *Alternatively*, if you are using an editor like VS Code, you can use extensions like **Live Server** to host it on a local environment.
